电赛磁悬浮控制系统

STM32F1 · 霍尔传感器 · PID 控制 · PWM 调流 · 闭环控制

项目背景

基于 STM32F1、霍尔传感器和电磁线圈搭建磁悬浮闭环控制系统。通过采集悬浮高度信号、 计算误差并输出 PWM 控制线圈电流，实现对悬浮物位置的动态调节。 项目覆盖传感器采样、PID 控制算法、PWM 驱动和示波器调试等嵌入式控制核心环节。

系统原理

/* 控制环路 */ 位置设定 ─► [误差计算] ─► [PID控制器] ─► PWM ─► [电磁线圈] ─► 悬浮物 ▲ │ └──────────────────────────────────────────────┘ 霍尔传感器 (位置反馈)

硬件组成

  • 主控：STM32F1
  • 传感器：霍尔传感器（位置反馈）
  • 执行器：电磁线圈 + PWM 驱动
  • 调试设备：示波器、万用表

关键难点与解决方案

1. PID 参数整定

围绕超调、振荡、稳态误差等典型控制问题进行参数调整。 先调 P（比例）消除响应慢的问题，再调 I（积分）消除稳态误差， 最后调 D（微分）抑制超调。

调试过程记录：
P 过大 → 系统振荡 → 减小 P
I 过大 → 积分饱和 → 限幅
D 过大 → 高频噪声放大 → 增加采样滤波

2. 传感器噪声处理

霍尔传感器输出存在噪声，直接送入 PID 会导致控制不稳定。 采用软件滤波（滑动平均 + 中值滤波）预处理传感器信号。

复盘总结

  • PID 不是玄学 — 理解 P/I/D 各自作用后，整定是有章可循的
  • 传感器滤波是控制的基础 — 噪声不处理好，PID 调再好也没用
  • 示波器是控制调试的眼睛 — 看到波形比看代码更能理解问题
